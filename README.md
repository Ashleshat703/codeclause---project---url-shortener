# URL Shortener

A simple and beginner-friendly **URL Shortener** web application built using **HTML, CSS, and JavaScript**. This project allows users to enter a long URL, generate a shortened version using the **shrtco.de API**, and copy the shortened link to the clipboard.

## Features

- Convert long URLs into short links
- Uses an external URL shortening API
- Copy shortened URL with one click
- Clean and simple user interface
- Beginner-friendly frontend project

## Tech Stack

- HTML5
- CSS3
- JavaScript
- shrtco.de API

## Project Structure

```text
url converter/
|-- unit.html
|-- style.css
|-- logic.js
|-- copy.js

How It Works
The user enters a long URL in the input box
JavaScript sends a request to the URL shortening API
The shortened URL is displayed in the output field
The copy button allows users to copy the shortened URL instantly
API Used
This project uses the public shrtco.de URL Shortener API to generate shortened links.

Getting Started
Clone or download this repository
Open the project folder
Open unit.html in your browser
Enter a valid URL and click the button to generate a short link
Example Use
Input: https://www.example.com/very-long-link
Output: A shortened URL generated by the API
